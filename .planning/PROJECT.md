# Oddscope

## What This Is

日本語の予測市場インテリジェンスプラットフォーム。Polymarketなどの海外予測市場データを日本語で整理・要約し、急変通知やAI分析をDiscordで届ける。市場を実際にトレードしている人が、朝のまとめチェックとリアルタイム通知の両方で情報をキャッチできるサービス。

## Core Value

予測市場の「今何が起きているか」を、日本語で瞬時に把握できること。

## Context

- Polymarket CLOBクライアントAPIが公開されており、市場データ・オーダーブック・取引履歴を取得可能
- ニュースAPIで関連ニュースを取得し、LLMで「なぜ動いたか」を下書き生成
- 日本語の予測市場情報源は現状ほぼ存在せず、ブルーオーシャン
- ターゲットユーザー: 予測市場トレーダー、情報収集層、データ利用者（v1はトレーダー優先）

## Constraints

- **Data Source**: v1はPolymarket APIのみ -- 安定性と深さを優先
- **Notification**: Discord -- ターゲットユーザーの利用率が高い
- **AI Cost**: LLMによる要約生成はコスト管理が必要 -- プレミアム機能として収益化
- **Tech Stack**: リサーチで決定（おまかせ）
