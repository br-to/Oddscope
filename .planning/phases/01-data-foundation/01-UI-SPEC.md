## Spacing Scale

Declared values (8-point grid system):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding |
| sm | 8px | Compact element spacing (table cell padding) |
| md | 16px | Default element spacing (form field gaps) |
| lg | 24px | Section padding (filter bar, table header) |
| xl | 32px | Layout gaps (between filter and table) |
| 2xl | 48px | Major section breaks (page header to content) |
| 3xl | 64px | Page-level spacing (top/bottom page margin) |

Exceptions: none

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 600 (semibold) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 600 (semibold) | 1.2 |

**Rationale**:
- Body 14pxは情報密度を高めつつ読みやすさを維持（テーブル表示に最適）
- ウェイトは2種類のみ（regular 400 / semibold 600）でシンプルに統一
- Labelも600を使用して視覚的な階層を明確化
- Line height 1.5で可読性を確保

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF | Background, main surface |
| Secondary (30%) | #F9FAFB | Table header, filter bar background |
| Accent (10%) | #3B82F6 | **Reserved for**: Filter submit button, theme tags, positive change indicators |
| Destructive | #EF4444 | Not used in Phase 1 (no destructive actions) |

**Accent reserved for**: フィルタ送信ボタン、テーマタグ（Badge）、24h変動がプラスの場合の表示色

**Neutral palette**:
- Text primary: #111827
- Text secondary: #6B7280
- Border: #E5E7EB
- Hover state: #F3F4F6

**Semantic colors**:
- Success (positive change): #10B981
- Warning (high volatility): #F59E0B
- Error (negative change): #EF4444

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | 絞り込み（フィルタ送信ボタン） |
| Empty state heading | 該当する市場が見つかりませんでした |
| Empty state body | フィルタ条件を変更するか、出来高の最小値を下げてみてください |
| Error state heading | データの読み込みに失敗しました |
| Error state body | ネットワーク接続を確認して、再試行ボタンを押してください |
| Error CTA | 再試行 |
| Loading state | 市場データを読み込み中... |

**Destructive actions**: Phase 1では該当なし（表示とフィルタリングのみ）

## Interaction States

### Table Row Hover
- Background: #F9FAFB
- Transition: 150ms ease-in-out
- Cursor: default（Phase 1では詳細ページなし、クリック不可）

### Filter Submit Button
- Default: bg-blue-600 text-white
- Hover: bg-blue-700
- Active: bg-blue-800
- Disabled: bg-gray-300 text-gray-500 cursor-not-allowed

### Badge (Theme Tags)
- Default: bg-blue-100 text-blue-800
- Hover: なし（インタラクティブ要素ではない）
